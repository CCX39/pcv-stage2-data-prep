# pcv-stage2-data-prep

本仓库用于 Work1 Stage2 后续真实数据准备、资产元数据整理与单帧 pilot 决策支持。

当前阶段仅进行外部资产、旧处理结果、导师脚本包与旧 XML/MPD 的只读审查。本仓库不在当前阶段实现真实切块、压缩、质量版本生成、Draco 编码、Stage2Input 批量生成或批量实验。

原始点云、批量 tile PLY、DRC、缓存、日志、压缩包和其他大体积资产不进入 Git。必要时仅在仓库中记录可审查的小型说明、审查报告或未来人工确认后的 manifest。

本仓库不修改既有 solver、距离标定项目、原始 Longdress 数据目录、旧处理结果目录或导师脚本包目录。

当前不提前承诺具体 tile 网格、PDL 规则、Draco 参数、MPD/XML 用途、decode-time 测量方法或最终 Stage2 数据格式。
